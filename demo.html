Welcome

I ripped this css test page off Paul Kilan for this test site, the original is here, the goal is to test classless css.

This css is based off system.css, I wanted to create a classless css that is inspired by System Software 6, but doesn't have complex css and a tonne of svg assets.

I have some basic javascript to toggle light/dark mode, the page will load with your system default.
